# Trabajo-de-Grado
Control mano robótica 

Control en lazo abierto de una mano robótica con comunicación via WiFi y interfaz visual de usuario.

"communication.py" - Establece una comunicación WiFi con una ESP32S3 como host local.
"GUI.py" - Configuración de la interfaz visual.
"hand.py" - Correr este script para abrir la GUI.
"motion_profiles" - Llena las pestañas de información sobre los datos enviados por la ESP32S3.
"styles.qss" - Encargada del estilo visual de la GUI.
